# Keyboard Shortcuts

mAI provides a comprehensive set of keyboard shortcuts so you can work without constantly reaching for the mouse. Master a few key ones and your workflow noticeably speeds up ŌĆö especially when switching Agents, managing conversations, or navigating the interface.

## How to Access Shortcut Settings

Click your user avatar in the top-right corner ŌåÆ **App Settings** ŌåÆ **Shortcuts**. You can view all shortcuts and customize most of them.

## Conversation Shortcuts

Conversation shortcuts use the **Option** key on Mac and **Alt** on Windows/Linux ŌĆö a different modifier from the Essential Shortcuts above.

| Action                          | Shortcut                                                                   |
| ------------------------------- | -------------------------------------------------------------------------- |
| **Open conversation settings**  | `Option + ,` (Mac) / `Alt + ,` (Win)                                       |
| **Regenerate message**          | `Option + R` / `Alt + R`                                                   |
| **Delete last message**         | `Option + D` / `Alt + D`                                                   |
| **Delete and regenerate**       | `Option + Shift + R` / `Alt + Shift + R`                                   |
| **New Topic**                   | `Option + N` / `Alt + N`                                                   |
| **Add message without sending** | `Option + Enter` / `Alt + Enter`                                           |
| **Edit message**                | `Option + double-click` message (Mac) / `Alt + double-click` message (Win) |

**Add message without sending** ŌĆö useful when you want to add context to the conversation without triggering an immediate response. The Agent will see the new message when it next replies.

## Custom Shortcuts

Most shortcuts can be customized. In the Shortcuts settings panel, click the shortcut you want to change, then press your desired key combination. Useful when:

- Your muscle memory conflicts with another app
- You want to prioritize actions you use most
- You're on a different keyboard layout

**Tip:** Start with the Command Menu (`Ōīś + K` / `Ctrl + K`) ŌĆö it's the fastest way to jump to any Agent, Topic, or setting.

## Use Cases

**Power-user workflow** ŌĆö `Ōīś + K` to open Command Menu ŌåÆ type Agent name ŌåÆ Enter. Switch Agents in under a second.

**Quick iteration** ŌĆö `Option + R` (Mac) / `Alt + R` (Win) to regenerate. `Option + Shift + R` / `Alt + Shift + R` if you want to delete the last message first, then regenerate.

**Multi-Agent work** ŌĆö Pin your top Agents to the sidebar, then use `Ctrl + 1ŌĆō9` to switch between them without touching the mouse.
