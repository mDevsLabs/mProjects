# mAI-1.2-Opal 💎

**mAI-1.2-Opal** est un assistant d'intelligence artificielle local, hautement fluide et multimodal. Optimisé pour s'exécuter avec **Ollama**, il offre un équilibre exceptionnel entre rapidité de réponse et capacités d'analyse textuelle et visuelle.

> **Objectif** : Mettre à disposition un modèle intermédiaire performant, capable d'analyser des documents visuels complexes et de traiter les tâches quotidiennes d'une entreprise. Productivité, codage, synthèse de documents, RAG et analyse d'images, le tout traité localement sans envoyer de données vers des serveurs tiers.

---

## ⚙️ Spécifications clés

- **Taille** : 33 milliards de paramètres (33B).
- **Contexte** : Fenêtre de contexte native de **256K tokens**.
- **Vision** : Prise en charge native de la vision multimodale.

---

## ✨ Fonctionnalités

- 💎 **Équilibre optimal** : Représente le compromis idéal entre rapidité d'exécution et puissance d'analyse.
- 📸 **Vision Multimodale** : Interprétation et analyse d'images, de diagrammes, de graphiques de données et de schémas.
- 🧠 **Assistance générale** : Formulation de réponses argumentées, structurées et aide à la rédaction de rapports professionnels.
- 🧾 **Synthèse de Documents** : Lecture et synthèse de documents longs, notes de réunion, articles de recherche ou fichiers logs.
- 💻 **Développement** : Génération et relecture de code dans de nombreux langages, aide à la recherche de bugs.
- 🔎 **Intégration RAG** : Prêt à s'intégrer avec un système de recherche documentaire interne pour interroger vos données locales.
- 🧰 **Appels d'outils (Tool Calling)** : Support de l'interaction avec des services tiers selon votre intégration locale.
- 🌍 **Support multilingue** : Français, anglais, espagnol, allemand, japonais, portugais, arabe, italien, coréen, chinois, etc.
- 🔐 **Garantie de confidentialité** : Exécution 100 % locale pour un respect absolu de la vie privée et de la propriété intellectuelle.

## 🚀 Guide d'installation rapide

### 1. Installer Ollama

Téléchargez et installez Ollama depuis la plateforme officielle :

[https://ollama.com/download](https://ollama.com/download)

### 2. Exécuter le modèle

Une fois Ollama installé, lancez le modèle dans votre terminal avec la commande suivante :

```bash
ollama run mDevsLabs/mAI-1.2-Opal
```
