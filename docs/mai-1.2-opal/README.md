# mAI-1.2-Opal 💎👁️

**mAI-1.2-Opal** est un assistant IA local, ultra-fluide et désormais **multimodal**. Conçu pour tourner parfaitement avec **Ollama**, il gère aussi bien tes textes que tes images.

> Objectif : t'offrir un modèle parfaitement équilibré, capable d'analyser tes visuels et de gérer tes tâches du quotidien. Productivité, code, résumé, RAG et analyse d'images — sans jamais envoyer tes données dans le cloud. Local-first, vibes clean 🧠✨

---

## ✨ Fonctionnalités

- 💎 **Équilibré & Fluide** : Le sweet spot parfait entre rapidité et intelligence pour ta machine.
- 📸 **Vision + Texte** : Balance tes images, screenshots ou graphiques, il les comprend et en extrait les infos direct.
- 🧠 **Assistant généraliste** : Questions/réponses, rédaction, reformulation, brainstorming efficace.
- 🧾 **Résumé & extraction** : Documents longs, notes, articles, logs et maintenant tes visuels.
- 💻 **Code** : Génération, explication, refactoring, aide au debug sans prise de tête.
- 🔎 **RAG-ready** : Utilisable avec un pipeline de recherche documentaire pour explorer tes propres fichiers.
- 🧰 **Tool/function calling-ready** : Prêt à s'intégrer avec des outils externes selon ton setup.
- 🌍 **Multilingue** : Français, anglais, espagnol, allemand, japonais, portugais, arabe, italien, coréen, chinois, etc.
- 🔐 **Local-first** : Fonctionne 100% en local via Ollama sur ta machine. Tes datas restent chez toi.
