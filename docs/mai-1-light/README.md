# mAI-1-Light ⚡

**mAI-1-Light** est un assistant IA local, léger et rapide, conçu pour tourner avec **Ollama** et propulsé par **IBM Granite 4.1 3B**.

> Objectif : offrir un modèle simple à lancer, utile au quotidien, orienté productivité, code, résumé, RAG et assistant généraliste — sans envoyer tes données dans le cloud. Local-first, vibes clean 🧠✨

---

## ✨ Fonctionnalités

- ⚡ **Léger** : basé sur Granite 4.1 3B, adapté aux machines modestes.
- 🧠 **Assistant généraliste** : questions/réponses, rédaction, reformulation, brainstorming.
- 🧾 **Résumé & extraction** : documents, notes, articles, logs.
- 💻 **Code** : génération, explication, refactoring, aide au debug.
- 🔎 **RAG-ready** : utilisable avec un pipeline de recherche documentaire.
- 🧰 **Tool/function calling-ready** selon l’intégration.
- 🌍 **Multilingue** : français, anglais, espagnol, allemand, japonais, portugais, arabe, italien, coréen, chinois, etc.
- 🔐 **Local-first** : fonctionne via Ollama sur ta machine.
